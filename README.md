a basic chatbot implementation using ChatOllama for generating responses and OllamaEmbeddings for taking in account of user's input documents all connected using langchain and deployed in website format through the steamlit framework.

models used: 

(1) llama3.2:latest

(2) nomic-embed-text:latest 

since ollama images can only run locally, making a docker container for the final deployment in huggingface,
the final website is viewable through the link mentioned in repository details.
